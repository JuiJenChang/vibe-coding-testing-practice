## [x] 【Mock API】登入失敗處理
**範例輸入**：
- 輸入有效 Email 和密碼
- Mock API 回傳 401 錯誤，訊息 "帳號或密碼錯誤"
- 點擊登入按鈕
**期待輸出**：
- 顯示全域錯誤橫幅，內容包含 "帳號或密碼錯誤"
- 登入按鈕恢復可點擊狀態

---

## [x] 【Mock API】登入成功跳轉
**範例輸入**：
- 輸入有效 Email 和密碼
- Mock API 回傳 200 成功
- 點擊登入按鈕
**期待輸出**：
- 呼叫 `login` 函式
- 頁面跳轉至 `/dashboard`
